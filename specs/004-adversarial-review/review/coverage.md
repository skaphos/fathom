## Scope notes

- **Anchor & drift**: reviewed at `cb845dd`. The v0.5.0 milestone was verified
  otherwise clear (only #217 open) on 2026-07-24, so no post-anchor product
  commits landed during the review. The three fix PRs (#252/#253/#254) target
  `main` and will be re-validated by CI on merge; they are not part of the
  reviewed anchor.
- **Perspective folding**: performance and docs were not run as standalone
  perspectives — performance concerns folded into correctness (bounded/idempotent
  reconcile) and docs-honesty into api-contract (Principle IX), rather than as
  separate passes. This is a deliberate scope bound for a release gate.
- **Tool assist**: `go -C tools tool task ci` and `task crd-compat` were run
  green at the anchor and seeded evidence, but no perspective consisted solely
  of tool output — each did a manual pass over its surface.
